# 📊 Estudo Completo de SQL para Análise de Dados

Repositório com um estudo consolidado em SQL baseado no curso **SQL para Análise de Dados e Data Science** da Data Science Academy (DSA).

![Status](https://img.shields.io/badge/Status-Em%20andamento-yellow)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-16-blue)
![Curso](https://img.shields.io/badge/Curso-DSA-orange)

## 🎯 Objetivo
Consolidar em um único arquivo SQL todos os conceitos estudados no curso, desde os fundamentos até técnicas avançadas, servindo como material de referência e portfólio profissional.

## 🛠️ Tecnologias
- PostgreSQL 16 e pgAdmin 4
- SQL (DDL, DML, DQL, DCL)

## 📂 Arquivo principal
- estudo_completo_sql.sql — Script único com todos os conceitos do curso comentados e organizados por tema.

## 📋 Conteúdo abordado
1. Criação e manipulação de banco de dados e tabelas (DDL)
2. Inserção, atualização e deleção de dados (DML)
3. Consultas e filtros (SELECT, WHERE, LIKE, BETWEEN, IN)
4. Funções de agregação (COUNT, SUM, AVG, MIN, MAX)
5. Agrupamento e ordenação (GROUP BY, HAVING, ORDER BY)
6. Categorização, Codificação e Binarização de Variáveis (CASE WHEN)
7. Junção de Tabelas (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, CROSS JOIN)
8. Window Functions (ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD, NTILE)
9. Subconsultas e CTEs (WITH)
10. Análise Exploratória de Dados com SQL
11. Limpeza e Transformação de Dados
12. Programação no Banco (Views, Stored Procedures, Functions, Triggers)
13. Otimização de Consultas (EXPLAIN, ANALYZE, Índices)
